Mathematics: natural numbers

Natural numbers: 0,1,2,3,4 ... and the principle of induction:

Induction Given a property P(n)...

  1. if P(0) is true, and
  2. whenever P(n) is true, then P(n+1) is true
  3. then in fact P holds for all numbers

The rules for reasoning about natural numbers can be formalized as e.g. https://en.wikipedia.org/wiki/Peano_axioms (axioms 1,6,7,8,9)

Proof

Rules are like:

Γ ⊢ A Γ ⊢ B
Γ ⊢ A ∧ B

Proofs are sequences (or trees) of things like rules. Here is a proof:

  1. A ⊢ A
  2. B ⊢ B
  3. A, B ⊢ A ∧ B // from 1 and 2, with weakening
  4. C ⊢ C
  5. A, B, C ⊢ A ∧ B ∧ C // from 3 and 4

Proofs are formal: a machine can handle the manipulation of these symbols and rules.

Limits of computation (4)

One of the very interesting results of computer science is:

Theorem It is impossible to write a program G(p,n) which, when given an arbitrary program p, and an input number n, can figure out whether the program p terminates on input n.

Termination is just one very simple property; effectively this theorem says that for any (non-trivial) property you like, constructing G is impossible.
